# 房产监控日报系统(云端全自动版)

## 核心特性
- 云端运行: GitHub Actions免费服务器,无需电脑开机
- 全自动: 每天定时采集并推送到手机
- 多数据源: 百度热搜 + 微博热搜 + 政府官网
- 政策监控: 中国政府网 + 住建部公告
- 完全免费: 0元成本
- 100%安全: 只访问公开页面,无风控

## 架构
```
GitHub Actions (定时触发)
    |
Ubuntu服务器运行Python
    |-- 百度热搜房产内容采集
    |-- 微博热搜房产话题采集
    |-- 中国政府网房产政策采集
    |-- 住建部公告采集
    |-- 生成日报
    |
手机微信 (每日推送)
```
